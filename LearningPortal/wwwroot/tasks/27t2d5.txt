Дана последовательность натуральных чисел. Расстояние между элементами последовательности  — это разность их порядковых номеров. Например, если два элемента стоят в последовательности рядом, расстояние между ними равно 1, если два элемента стоят через один  — расстояние равно 2 и т. д. Назовём парой любые два числа из последовательности. Необходимо определить количество пар, в которых сумма элементов и расстояние между ними имеют равные остатки от деления на 9.
Первая строка входного файла содержит целое число N    общее количество чисел в наборе. Каждая из следующих N строк содержит одно число, не превышающее 109.
Вам даны два входных файла (A и B), каждый из которых имеет описанную выше структуру. В ответе укажите два числа: сначала искомое количество пар для файла A, затем  — последние 6 цифр искомого количества пар для файла B.
Ответ: 346204 407867.
